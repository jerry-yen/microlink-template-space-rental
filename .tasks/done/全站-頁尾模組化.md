# 全站-頁尾模組化 (Site-wide - Footer Modularization)

## 1. 任務目標 (Objective)
- **目的**：將全站 PHP 頁面的頁尾 (Footer) 區塊模組化，統一管理於 `include/footer.php`，以利後續維護與修改。
- **涉及檔案**：目錄下所有包含 Footer 內容的 PHP 檔案。

## 2. 執行清單 (Action Items)
- [x] 檢查目錄下所有 PHP 檔案，找出含有 `<footer>` 標籤的檔案。
- [x] 在 `include` 目錄下新增 `footer.php` 檔案。
- [x] 將原本的頁尾 HTML 內容提取至 `include/footer.php`。
- [x] 將原本 PHP 檔案中的 `<footer>...</footer>` 區塊取代為 `<?php include dirname(__FILE__) . '/include/footer.php'; ?>`。
- [x] 確認所有頁面顯示正常。

## 3. 驗證重點 (Verification)
- [x] 首頁及各內頁的頁尾顯示是否一致且正常。
- [x] 確保無語法錯誤。
