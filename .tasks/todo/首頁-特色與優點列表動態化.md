# 簡易任務 (Task - Simple)

**請逐一執行以下步驟，每執行一步，就標記為完成，直到所有步驟都完成。完成執行清單之後也請進行驗證，逐一進行驗證，直到所有驗證都通過。完成後，請在第一大標題後面加上「 - 現在日期時間」的格式。**

## 1. 任務目標 (Objective)
- **目的**：將首頁「特色與優點」區塊原本寫死的 8 個靜態按鈕，改為讀取系統中的場地設備或服務清單，以動態呈現提供的服務項目。
- **涉及檔案**：`microlink-template-space-rental/index.php`

## 2. 執行清單 (Action Items)
- [ ] 確認後台傳遞的設備或服務清單變數 (例如 `$equipments` 或 `$services`)。
- [ ] 修改 `index.php` 的「特色與優點」區塊，移除寫死的 `col` 項目。
- [ ] 使用 `foreach` 迴圈遍歷清單，動態產生對應的按鈕與 Icon (若後台無 Icon 設定，可需設計對應機制或使用預設圖示)。
- [ ] 確保排版維持 RWD (手機版 2 欄 / 桌機版 4 欄) 的 Grid 佈局。

## 3. 驗證重點 (Verification)
- [ ] 前台顯示的特色/設備列表內容與後台資料一致。
- [ ] 數量變動時 (例如從 8 個變為 5 個或 10 個)，版面排列依然整齊。
