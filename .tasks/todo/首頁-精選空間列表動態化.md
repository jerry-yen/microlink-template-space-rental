# 簡易任務 (Task - Simple)

**請逐一執行以下步驟，每執行一步，就標記為完成，直到所有步驟都完成。完成執行清單之後也請進行驗證，逐一進行驗證，直到所有驗證都通過。完成後，請在第一大標題後面加上「 - 現在日期時間」的格式。**

## 1. 任務目標 (Objective)
- **目的**：將首頁「精選空間照片」區塊的 6 張靜態卡片，改為讀取後台建立的空間資料列表，讓使用者能瀏覽真實的場地資訊並點擊進入詳情頁。
- **涉及檔案**：`microlink-template-space-rental/index.php`

## 2. 執行清單 (Action Items)
- [ ] 確認後台傳遞的空間列表變數 (例如 `$spaces` 或 `$featured_spaces`)。
- [ ] 修改 `index.php` 的空間列表區塊，保留一份卡片 HTML 結構作為樣板，移除其餘靜態內容。
- [ ] 使用 `foreach` 迴圈遍歷空間資料，動態填入以下資訊：
    - [ ] 圖片 (`src`)
    - [ ] 空間名稱 (`card-title`)
    - [ ] 簡述 (`card-text`)
    - [ ] 容納人數 (`capacity`)
    - [ ] 地點 (`location` 或 `city`)
    - [ ] 詳情頁連結 (改為 `space.php?id={id}`)
- [ ] 處理無資料或圖片缺失時的預設顯示 (Placeholder)。

## 3. 驗證重點 (Verification)
- [ ] 空間卡片正確顯示後台建立的空間資訊。
- [ ] 點擊「查看更多」或卡片本身，能正確跳轉至對應的 `space.php` 詳情頁。
- [ ] 版面在不同筆數下 (例如 1 筆或 10 筆) 排列是否正常。
